# Fiducial-3D-Mapper

## Installation
### MacOS
```zsh
$ git clone https://github.com/mmatl/pyglet.git
$ cd pyglet
$ pip install .
```

### Other Platforms
```bash
$ pip install pyrender
```
## Run
`$ python3 main.py`

### Keyboard Inputs at Render Window
    q: quit
    a: toggle draw axis
    b: toggle draw box
    b: toggle draw bunny
